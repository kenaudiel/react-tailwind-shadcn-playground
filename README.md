# About
This is a repository served as a personal playground for building mock-up frontend website using modern frameworks & libraries. I believe hands-on coding is the most effective way to learn new things that could stick in long-term. The stack includes:
- **React.js**
- **Tailwind CSS**
- **shadcn/ui**
etc.
